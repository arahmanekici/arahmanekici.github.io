# TÜGVA Logo Sertifika Oluşturucu

Bu web uygulaması ile TÜGVA logo şablonuna metin ekleyerek sertifika oluşturabilirsiniz.

## Özellikler
- Museo900 fontu ile 41 punto metin ekleme
- HEX #00B6C5 rengi
- Yatayda ortalanmış metin
- PDF ve PNG çıktı
- Şeffaf arka plan

## Kullanım
1. Metin girin (ör. KAYSERİ)
2. "Sertifika Oluştur" butonuna tıklayın
3. `logo_METIN.pdf` ve `logo_METIN.png` dosyaları indirilecek

## Gerekli Dosyalar
- `logo.pdf`: Orijinal logo şablonu
- `Museo900.ttf`: Özel font dosyası
